All keyboard shortcuts available in Fury. On Windows and Linux, replace **Cmd** with **Ctrl**.

## General

| Shortcut | Action |
|----------|--------|
| **Cmd+K** | Open command palette |
| **Cmd+N** | Create new workspace |
| **Cmd+,** | Open settings |
| **Cmd+B** | Toggle right sidebar |
| **Escape** | Close current panel/dialog |

## Navigation

| Shortcut | Action |
|----------|--------|
| **Cmd+1** | Files panel |
| **Cmd+2** | Changes panel |
| **Cmd+3** | Checks panel |
| **Cmd+J** | Focus terminal |
| **Cmd+Shift+F** | Search workspaces |

## Chat

| Shortcut | Action |
|----------|--------|
| **Enter** | Send message |
| **Shift+Enter** | New line in message |
| **Option+T** | Toggle thinking mode |
| **Option+P** | Open model selector |
| **Shift+Tab** | Toggle plan mode |
| **Cmd+Shift+Enter** | Approve plan/permission |

## Editor

| Shortcut | Action |
|----------|--------|
| **Cmd+S** | Save file |
| **Tab** | Accept Copilot suggestion |

## Integrations

| Shortcut | Action |
|----------|--------|
| **Cmd+I** | Link Linear issue |
