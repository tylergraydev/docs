---
title: Linear Integration
description: Search and link Linear issues to workspaces
---

Fury integrates with [Linear](https://linear.app) for issue tracking. Link issues to workspaces to keep context about what you're working on.

## Setup

1. Go to **Settings > Linear**
2. Enter your Linear API key (get one from Linear's settings under API)
3. Save

## Searching issues

Press <Tooltip tip="Ctrl+I on Windows/Linux">**Cmd+I**</Tooltip> to open the issue search. Type to search across your Linear issues by title or identifier.

Results show:

- Issue identifier (e.g., `ENG-123`)
- Title
- Status
- Team
- Priority

## Linking issues

Select an issue from the search results to link it to the current workspace. Linked issues appear in the workspace panel, providing context about the task at hand.

## Unlinking issues

Remove a linked issue from the workspace panel when it's no longer relevant.

## How it helps

Linked issues give the agent additional context. When you start a conversation, the agent can see what issue you're working on and tailor its approach accordingly.
