---
title: Command Palette
description: Quick actions with Cmd+K
---

The command palette provides fast access to common actions without navigating menus.

## Opening

Press <Tooltip tip="Ctrl+K on Windows/Linux">**Cmd+K**</Tooltip> to open the command palette.

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| <Tooltip tip="Ctrl+K">**Cmd+K**</Tooltip> | Command Palette |
| <Tooltip tip="Ctrl+N">**Cmd+N**</Tooltip> | New Workspace |
| <Tooltip tip="Ctrl+B">**Cmd+B**</Tooltip> | Toggle Right Sidebar |
| <Tooltip tip="Ctrl+J">**Cmd+J**</Tooltip> | Focus Terminal |
| <Tooltip tip="Ctrl+,">**Cmd+,**</Tooltip> | Settings |
| <Tooltip tip="Ctrl+S">**Cmd+S**</Tooltip> | Save File |
| <Tooltip tip="Ctrl+1">**Cmd+1**</Tooltip> | Files Panel |
| <Tooltip tip="Ctrl+2">**Cmd+2**</Tooltip> | Changes Panel |
| <Tooltip tip="Ctrl+3">**Cmd+3**</Tooltip> | Checks Panel |
| <Tooltip tip="Ctrl+Shift+F">**Cmd+Shift+F**</Tooltip> | Search Workspaces |
| **Option+T** | Toggle Thinking Mode |
| **Option+P** | Model Selector |
| **Shift+Tab** | Toggle Plan Mode |
| <Tooltip tip="Ctrl+I">**Cmd+I**</Tooltip> | Link Linear Issue |
| <Tooltip tip="Ctrl+Shift+Enter">**Cmd+Shift+Enter**</Tooltip> | Approve Plan/Permission |
| **Escape** | Close panel/dialog |

On Windows and Linux, use **Ctrl** instead of **Cmd**.
