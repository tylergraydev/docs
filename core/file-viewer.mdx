---
title: File Viewer
description: Browse, read, and edit files with Monaco Editor
---

Fury includes a built-in file viewer and editor powered by [Monaco Editor](https://microsoft.github.io/monaco-editor/) — the same editor that powers VS Code.

## Browsing files

Open the **Files** panel (<Tooltip tip="Ctrl+1 on Windows/Linux">**Cmd+1**</Tooltip>) to see a tree view of all files in your workspace. Click any file to open it in a tab.

## Editing files

Edit files directly in Fury with full syntax highlighting for 100+ languages. Save with <Tooltip tip="Ctrl+S on Windows/Linux">**Cmd+S**</Tooltip>.

When format-on-save is available (e.g., Prettier is configured in your project), files are automatically formatted when saved.

## File tabs

Open multiple files in tabs. Unsaved changes are indicated with a dot on the tab.

## GitHub Copilot completions

When [Copilot](/core/copilot) is enabled, you get inline code completions as you type in the editor.

## Type definitions

For TypeScript projects, Fury automatically loads type definitions to provide accurate IntelliSense and error highlighting in the editor.
